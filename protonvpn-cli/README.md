# protonvpn-cli

## About

Proton VPN client without a graphical interface. Useful for integration with
other tools. The replacement client is graphical-only and does not offer a CLI.

Depends on:
- [python-protonvpn-nm-lib](../python-protonvpn-nm-lib/)
